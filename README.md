# Решение двойного интеграла

Вычислим следующий двойной интеграл:

$$\int_0^\infty \int_0^{\pi/2} \frac{x \cdot \sin(\theta) \cdot \ln(1 + x^2 \cos^2(\theta))}{(1 + x^2 \sin^2(\theta))^{3/2}} d\theta \, dx$$

## Шаг 1: Замена переменных

**Замена:** $u = x\sin(\theta)$, $v = x\cos(\theta)$

**Почему:** Эта замена естественна для полярных координат, она упрощает структуру интеграла, разделяя переменные в числителе и знаменателе.

Якобиан преобразования: $du \, dv = x \, d\theta \, dx$ (из геометрии), также $u^2 + v^2 = x^2$.

Получаем:

$$\int_0^\infty \int_0^\infty \frac{v \cdot \ln(1 + v^2)}{(1 + u^2)^{3/2}} \cdot \frac{du \, dv}{u^2 + v^2}$$

## Шаг 2: Интегрирование по u

$$\int_0^\infty \frac{du}{(u^2 + v^2)(1 + u^2)^{3/2}} = \frac{\pi}{2v^2\sqrt{1 + v^2}}$$

**Почему:** После замены переменных интеграл разделился - $v$ входит только как параметр в интеграл по $u$. Этот интеграл вычисляется стандартными методами (разложение на простые дроби или таблицы).

## Шаг 3: Оставшийся интеграл

$$\frac{\pi}{2} \int_0^\infty \frac{\ln(1 + v^2)}{v \sqrt{1 + v^2}} dv$$

**Почему:** Подставляем результат из шага 2, упрощаем и получаем одномерный интеграл.

## Шаг 4: Тригонометрическая замена

**Замена:** $v = \tan(t)$

$$\frac{\pi}{2} \int_0^{\pi/2} \frac{\ln(\sec^2(t))}{\tan(t)} dt = \pi \int_0^{\pi/2} \frac{\ln(\sec(t))}{\tan(t)} dt$$

**Почему:** Замена $v = \tan(t)$ превращает $\sqrt{1 + v^2} = \sec(t)$, что упрощает корень. При этом $dv = \sec^2(t) \, dt$, и $\ln(1 + v^2) = \ln(\sec^2(t)) = 2\ln(\sec(t))$.

## Шаг 5: Известный результат

$$\int_0^{\pi/2} \frac{\ln(\sec(t))}{\tan(t)} dt = \frac{\pi}{2}$$

**Почему:** Это классический интеграл, который вычисляется через разложение в ряд или контурное интегрирование. Он встречается в теории специальных функций.

## Окончательный ответ

$$\boxed{\frac{\pi^2}{2}}$$
